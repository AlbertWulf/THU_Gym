-------------主要功能------------------------------------------------------------------------
选择特定时间预定特定场次，如选择明天早上八点预约气膜馆羽2 20:30-22:30场次，预约之后15分钟内需要付款，当然可以选择pay way为2，为线下付款
------------------------------------------------------------------------------------------
交互比较原始，时间有限，login.py可以进行交互，不过不推荐使用，还是在main.py中直接修改各项id
------------------------------------------------------------------------------------------
Version 0.1
------------------------------------------------------------------------------------------
至主函数修改:main.py
login_post_data中'un'为学号，'pw'为密码
-------------------------------
Air_film_gym_book_data中：
bookData.book_person_phone ：info预留手机号
gymnasium_idForCache：不同体育馆对应id不同，请至data.py寻找对应id
item_idForCache:请至data.py寻找
allFieldTime:为场地id+"#"+时间格式，如“4045968#2020-10-30”
场地id 首先登陆预约界面，在想选择的场次（无需特定日期）上鼠标右击，选择检查元素，得到的resource_id即为该位置选择时间段对应的id
--------------------------------
倒数第二行，scheduler.add_job中修改hour 和minute
houre为24小时制，如下午1点应该输入13
----------------------
运行：
win+R 输入cmd 打开终端，cd 至本文件目录，执行python main.py，保持联网状态，则会在既定日期和时刻
